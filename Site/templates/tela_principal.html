ANALISANDO ATRAVÉS DE VÍDEOS.

O site funciona como uma biblioteca virtual, Guardando vídeos diversos para análise e estudo. Nele, os vídeos são cuidadosamente arquivados, Prontos para serem explorados e analisados.

NOSSOS OBJETIVOS.

Biblioteca

Uma biblioteca capaz de armazenar os vídeos de drone gravados pelo usuário, possuindo sistema de gerenciamento e armazenamento na nuvem.

Analise

Uma IA para análise de vídeo capaz de detectar e identificar objetos.

Comunidade

Planejamento para integrar uma comunidade online, oferecendo um espaço virtual para os membros se conectarem, compartilharem recursos e colaborarem em projetos.
